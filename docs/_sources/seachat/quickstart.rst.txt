Introduction
============

This SeaChat tutorial provides a quick overview on how to access and navigate
SEASALT.AI_'s SeaChat_ platform.

By the end of this tutorial, you will be able to:
  * Access SeaChat
  * Create a Project
  * Orient yourself in the VSCode Editor

Accessing SeaChat
=================

SeaChat is hosted by SEASALT.AI and can be accessed using any web browser.

**To Access:**

1. Go to `<https://seachat.seasalt.ai>`_
